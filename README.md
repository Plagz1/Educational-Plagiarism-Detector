# Educational Plagiarism Detector (IT Service)

Проект по дисциплине **«Информационные технологии и сервисы»**.

## Описание
Educational Plagiarism Detector — это информационный сервис для автоматического
анализа текстовых учебных работ на предмет заимствований с использованием
современных методов обработки текстов и облачных сервисов CI/CD.

## Цель проекта
Создание программного сервиса, обеспечивающего:
- автоматизацию анализа текстовых данных;
- применение алгоритмов сравнения документов;
- использование сервисов непрерывной интеграции;
- формирование машиночитаемых отчётов.

## Основной функционал
- Загрузка файлов формата TXT и PDF
- Предварительная обработка текста
- Сравнение документов:
  - TF-IDF + Cosine Similarity
  - Longest Common Subsequence
  - N-gram matching
- Формирование JSON-отчётов с временной меткой
- Визуализация матрицы похожести
- Автоматический запуск анализа через CI/CD

## Используемые информационные технологии
- Python (обработка данных)
- Scikit-learn (анализ текстов)
- GitHub Actions (облачный CI/CD сервис)
- JSON (формат обмена данными)
- PyTest (контроль качества)

## CI/CD
Реализованы следующие сценарии:
- автоматическая проверка кода при каждом коммите;
- плановый запуск анализа по расписанию;
- ручной запуск workflow;
- сохранение результатов как артефактов.
